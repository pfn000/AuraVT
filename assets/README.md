# Assets

This folder contains application icons and resources.

## Required Icons

Before building for production, add the following icon files:

| File | Platform | Size |
|------|----------|------|
| `icon.ico` | Windows | 256x256 (multi-resolution) |
| `icon.icns` | macOS | 1024x1024 |
| `icon.png` | Linux/General | 512x512 or 1024x1024 |

### Creating Icons

1. **Start with a 1024x1024 PNG** with transparency
2. Use online converters or tools:
   - Windows ICO: https://icoconvert.com/
   - macOS ICNS: https://iconverticons.com/online/
3. Place files in this folder
